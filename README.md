# found-ip-by-domain-excel

Quickly search for IPs from domain names listed on excel.

Search the example.xlsx domains and export to result.xlsx the IPs.

## Requirements

* Python version 3.6

## How to use

```
python3 -m venv ipvenv
source bin/activate
pip install -r requirements.txt
python run.py
```
